# RPP MI ISLAMIYAH Premium

Sistem administrasi guru, supervisi, dan dashboard analitik terpadu untuk MI ISLAMIYAH Malang.

## Fitur Utama
- **Modul RPP (5 Tab)**: Identitas, Identifikasi, Desain, Pengalaman, Asesmen.
- **Modul LKPD (3 Tab)**: Identitas, Aktivitas, Refleksi.
- **Rubrik Penilaian**: Kalkulasi nilai otomatis & predikat.
- **Supervisi**: Panel khusus Supervisor untuk review RPP.
- **Dashboard Kepsek**: Analitik grafik aktivitas, nilai, dan kehadiran.
- **Manajemen Guru**: CRUD data guru dan status aktif.

## Teknologi
- Tailwind CSS (UI/UX)
- Alpine.js (Logic & Reactivity)
- Chart.js (Data Visualization)
- LocalStorage (Persistence)
